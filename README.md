# KillProcGui
GUI for [KillProc](https://github.com/Cutotopo/killproc)

## Prerequisites
 - Python3
 - PyQT 5

## Configuring PyQt
Create a venv:

`python3 -m venv venv && source venv/bin/activate`

Install PyQt5:

`pip install PyQt5`

Run the python file:

`python3 __main__.py`

## Launching
`cd <repo location> && source venv/bin/activate && python3 __main__.py`

or zip the repo folder and use

`python3 zip_path`
